# Glass Core Substrate

> **Demand driver.** Shift from organic substrates to glass-core substrates for high-layer-count AI GPU/ASIC advanced packaging.

**What it is:** Glass core substrates enabling more interconnect layers (7→11→20+) for AI compute packaging; bottleneck is glass mechanical performance under thick multi-layer copper stress.
